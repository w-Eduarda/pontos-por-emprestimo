# Casos de Uso / User Stories

## UC01 – Cadastrar Usuário
**Ator:** Visitante  
**Descrição:** Permite que um novo usuário crie conta na plataforma  
**Pré-condição:** Não estar cadastrado  
**Pós-condição:** Conta criada e ativa

## UC02 – Emprestar Recurso
**Ator:** Usuário autenticado  
**Fluxo Principal:**
1. Usuário seleciona recurso para disponibilizar
2. Sistema registra empréstimo e adiciona pontos
