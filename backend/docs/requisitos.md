# Documento de Requisitos do Sistema

## 1. Introdução
### 1.1 Objetivo
Descrever os requisitos funcionais e não funcionais da plataforma de compartilhamento de recursos.

### 1.2 Escopo
O sistema permitirá que usuários emprestem e solicitem recursos usando um sistema baseado em pontuação.

## 2. Definições / Glossário
| Termo | Definição |
|-------|-----------|
| Recurso | Item ou serviço disponível para empréstimo |
| Pontos | Sistema de recompensa e consumo |

## 3. Requisitos Funcionais
| ID | Descrição | Prioridade |
|----|-----------|------------|
| RF01 | O usuário deve poder cadastrar-se na plataforma | Alta |

## 4. Regras de Negócio
| ID | Descrição |
|----|-----------|
| RN01 | Cada empréstimo realizado gera pontos para o doador |

## 5. Requisitos Não Funcionais
| ID | Tipo | Descrição |
|----|------|-----------|
| RNF01 | Usabilidade | Interface intuitiva |
